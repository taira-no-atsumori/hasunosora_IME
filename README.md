# 概要
蓮ノ空関連のIME(辞書)がなかったみたいなので作りました。  
Google IMEのみ対応しています。  
その他のIMEへの対応予定は今のところありません。  

# 注意事項
## 改変
改変はOKです。  
むしろ、これをベースにみなさんが使いやすいように変えちゃってください。  

## 二次配布(再配布)
このIMEを不特定多数の方にそのまま二次配布(再配布)するのはNGです。  
ただし、これをベースに改変したものを配布するのはOKです。  
その際は、このページのリンクとこれをベースにしたことがわかるような記載をお願いします。
